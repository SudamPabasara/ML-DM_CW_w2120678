# ML-DM_CW_w2120678
Project Overview: Loan Processing Automation & AnalysisThis project, part of the Machine Learning & Data Mining module (5DATA002W.2) at the University of Westminster, tasks you with acting as a Data Scientist within a financial institution. Working alongside bankers and risk analysts, your primary goal is to leverage historical consumer data to automate and optimize two critical aspects of the lending process: loan approval and maximum lending value estimation.
The project is structured around the CRISP-DM methodology (excluding deployment) and requires a professional, concise approach to technical analysis.

Core Objectives
The company relies on your machine learning models to answer two key business questions:
Automation potential: Can machine learning accurately automate the decision-making process for loan approvals? 
Reliable estimation: For approved clients, can models provide a dependable estimate of the maximum amount the bank should lend to maximize returns while minimizing risk?
